# crates/bridge — Magnus Bridge Standard (MBS)

This directory is intentionally empty. It is the designated home for the
cross-chain settlement bridge: routing tokens, BLS threshold certificates,
bridge contracts on destination chains (Ethereum, Tron, Solana, BNB,
etc.), and the MagnusBridge.sol template.

Implementation is a follow-up spec. Do not add code here as part of the
Tempo → Magnus rename refactor.

# crates/payments — Payment Engine

This directory is intentionally empty. It is the designated home for the
native Payment Engine subsystem of the Magnus L1: typed payment operations
(`Transfer`, `PathPayment`, `Escrow`, `ScheduledPayment`), the FX order
book, settlement tiers, and the multilateral netting engine.

Implementation is a follow-up spec. Do not add code here as part of the
Magnus rename refactor.

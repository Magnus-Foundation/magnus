# crates/gateway — Magnus Gateway Protocol (MGP)

This directory is intentionally empty. It is the designated home for the
fiat gateway protocol: the MGP specification, on/off-ramp lifecycle,
gateway registry, fiat rail integrations (VietQR, M-Pesa, UPI, PIX,
etc.), and settlement attestation.

Implementation is a follow-up spec. Do not add code here as part of the
Tempo → Magnus rename refactor.
